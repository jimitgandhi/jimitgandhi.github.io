Projects

My Projects

I have worked on several projects ranging from full-length research papers to android applications for my college. A few of these projects were:

SMS Text Normalization

A pipeline structure to convert SMS language to regular language that involves acronym expansion, named entity recognition, dictionary matching, and masking and prediction of words using the BERT model. Co-authored a paper on the same.

YOLO based Recognition of Indian License Plates

A unique model to detect Indian License Plates using transfer learning. Includes the creation of a custom dataset to train the model uniquely on Indian plates. Published and presented a paper on the same at ICACTA-2020.

Image Modification using Text with GANs

A deep-learning based model that uses novel approach to perform Language-Based Image Editing, involving the use of text adaptive discriminators, bilinear residual layers, and various contextual word representations.

Sign Language Recognition

A model created using Logistic Regression, SVM, Artificial Neural Networks and LSTM models to predict hand signs. The dataset used for training was collected using non-intrusive approaches.

Stock Manager

A web application to manage stocks traded in the market with real-time predictions made using an RNN network. It is complete with several features such as a mock trading app, forex transactions, news module and a chatbot.

Political Sentiment Analysis

A web application to analyse the sentiments of the public for various trending news topics. The analysis is done using data from several social media websites like Quora and Twitter.
